# Task 1 — Google Tag Manager Event Schema

This folder contains the event tracking design for the booking flow, built for Google Tag Manager (GTM) and Google Analytics 4 (GA4).

## Contents

- [event-schema.md](./event-schema.md) — full list of tracked events and their parameters
- [booking-funnel.md](./booking-funnel.md) — step-by-step breakdown of the booking funnel
- [datalayer-json.md](./datalayer-json.md) — sample `dataLayer` payloads for each event
- [ga4-funnel.md](./ga4-funnel.md) — how the funnel is set up and reported on in GA4

## Goal

Track user behaviour through the booking funnel (landing → service selection → booking form → confirmation) in a way that is consistent, reusable, and easy for GTM/GA4 to consume.
