# Booking Funnel

This document contains the booking funnel breakdown for Task 1.

## Funnel Steps

1. **Landing** — user arrives on the healthcare landing page
2. **Service Selection** — user browses and selects a service
3. **Booking Form** — user fills in their details and submits
4. **Confirmation** — booking is confirmed and a success message is shown

## Notes

Each step above maps to one or more events in [event-schema.md](./event-schema.md). The funnel will later be visualised in GA4 using an Exploration report (see [ga4-funnel.md](./ga4-funnel.md)).
